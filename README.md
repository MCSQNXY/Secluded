# Windows
- 下载解压 找到对应 系统文件夹 x64 或者 x86 找到里面的 SecludedLauncher.exe 双击运行即可
- 进入主界面 有 账号列表 词库列表 日志列表 运行设置 四个页面

# Linux
- 下载解压 找到对应 系统文件夹 x64 进入 使用 cd 命令 切换工作目录
- 切换工作目录 cd x64
- 给予权限 chmod +x SecludedLauncher.out.sh
- 运行程序 ./SecludedLauncher.out.sh
